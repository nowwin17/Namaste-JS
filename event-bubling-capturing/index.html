Event Bubbling and Event Capturing(trickling)

You can see there is three div in heirarchy and attached click event to all of them.

document.querySelector("#") .addEventListener('click',()=>{ console.log("clicked"); }, );

we can add event listener on click so it has three parameters

  1. Event (in this case its 'click'
  2. Callback function(when event occurs it will execute function)
  3. Boolean(use capture) value(It will decide hows event propogating(bubling or capturing))

If we not passed any value to boolean then it takes default value i.e. false

document.querySelector("#") .addEventListener('click',()=>{ console.log("clicked"); });

If it is false then event propogating like down to up which is called Event Bubbling

If we passed 'true' value to boolean

document.querySelector("#") .addEventListener('click',()=>{ console.log("clicked"); }, true);

If it is true then event propogating like up to down which is called Event Capturing or Event Trickling
