Call, Apply Bind Methods


Call

Binds "this" value, invokes the function and allow us to pass list of aguments

e.g.

            
              function test(arg1, arg2){
                console.log(this.num, arg1, arg2);
              }
              var obj = { num: 100 };
              test.call(obj, 10, 20);

Apply

Bind "this" value, invokes the function and allow us to pass list of aguments as an array

e.g.

            
              function test(...arg){
                console.log(this.num, arg);
              }
              var obj = { num: 100 };
              test.apply(obj, [10, 20,30]);

Bind

Binds "this" value, returns a new function and allow us to pass list of aguments

e.g.

            
              function test(arg){
                console.log(this.num, arg);
              }
              var obj = { num: 100 };
              let bindedFn= test.bind(obj,"arguments");
              bindedFn()

All three methods set "this" argument to the function

->The Call & Apply methods set "this" to function & call function

->The Bind method will only set "this" to function and we will need to seperately invoke function
